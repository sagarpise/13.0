Integrate WhatsApp with Odoo for Live Chat

1.5 billion users in 180 countries makes WhatsApp the most-popular messaging app in the world.
Now, you can assist your customers directly via WhatsApp with the help of Odoo WhatsApp Chat Integration. The module adds WhatsApp chat widget in Odoo website so that your customers can message you.
Moreover, you can segregate teams to assist the website visitors so that they can chat with the concerned person. It will save your time as well as your customers time.

WhatsApp Chat Head in Odoo Website

WhatsApp chat head is added in the Odoo Website so that the customers can easily find a way to connect to you.
They can simply chat to the team for the kind of support they require.

Set Customized Layout For WhatsApp Chat Widget

You can completely customize the WhatsApp chat widget for your Odoo website.
So, if you are using any theme for your Odoo Website then you can customize the WhatsApp chat wizard as per the color of your theme to improve the outlook.
