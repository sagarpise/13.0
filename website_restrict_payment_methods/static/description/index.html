Website Shop - Restrict Payment Acquirer Methods - Odoo App

Payment Methods / Gateway Restriction to Customers on Ecommerce Odoo Shop

This app allows to restrict payment acquirer methods for specific customers from the website.


Main Features:

  • You can configure restricted payment acquirers on customer form so when the customer login then restricted payment acquirers will not be visible to customers to choose on shop payment page.
  • Customer specific restrictions can be applied.
  • If you do not set a restricted payment method on customer form then for that customer all payment methods will be visible as per Odoo standard.
